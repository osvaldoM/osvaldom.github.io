registry

organization that manages top level domains like .com, .io or .org they also seet rules on domain names such as extensions, registration process and work with registars on selling domain names. veriSign is the registry for .com for instance

registars

organizations like @mozdomains and @iberweb that sell domain names to customers(registrants) they are responsable for associating a domain name to a customer and then sending or updating this information to the registry's database.

registrant

people who buy a domain name

How it goes

  • client makes a request for www.address.com
  • user-agent makes a dns lookup
  • dns server forwards request to nameserver
  • nameserver matches domain name with an ip address or another domain name using records
  • ip address is returned
  • user agent makes request to server ip
